# Archipelago Manual for War Thunder
War Thunder is an a Free-to-play Online Team fighter, where players take control of various vehicles from different (world) wars.
Vehicles include: Planes, Helicopter, Tanks, Ships

This AP-Implementation can take a long time to beat, depending on the goal.
As a single round lasts around 30 minutes, this is best experienced in an Async.

# Requirements
- War Thunder (From Steam or Official Website)
- A long played account (New account support technically there, but not with best implementation)
- Latest Release of the Manual-AP-World for War Thunder
- A lot of time

# Current Features
- Unlockbale Progressive and separate BR (Battle Rating)
- Unlockable and excludable Nations
- Unlockable and excludable Vehicles (Currently only Aviationa nd Ground Targets)
- Multiple Goals (Kill on every BR, with every Nation)
- Logic inclusion for Premium and Old Vehicles
  - Old Vehicles mean those that are (currently) no longer availbale
- Ability to swtich between simple and complex Vehicle Types
  - E.g.: Instead of "Aviation"
- Partial Implementation for Unlockable Upgrades (Very WIP - Currently Aviation Only)

# Restrictions & Rules
1. You are only allowed to play with those Vehicles and Nations on the BRs, that you have unlocked.
2. Checks that are named like "China 5.X (Aviation)" require to defeat another player with a Plane from China on the BR 5 (5.0, 5.3 or 5.7 or whatever they're named).
3. You must keep track of "Wins", "Kills" and "Assists" on your own.
4. A "Win" is determined by staying alive until end of the round and winning the Round.
  4.1. If you get defeated and sent back to the Hangar, but your Team Wins in post, that will not count!
5. "Include Premium" can logically expect you to spend actual money! Only activate it, if you already have done so before, or are fine with it!
6. "Include Old Vehicles" can logically expect you to be required to own vehicles you can no longer obtain! Only activate it if you're a hardcore collector!

# Troubleshooting
- If you should stumble into issues regarding logic or with the APWorld, please make an Issue or write it down in the Manual-Discord Forum Post!
- If Logic expects you to maneuver a vehicle that you haven't unlocked in-game yet, then either unlock it first, or player another vehicle from the same Nation.
  - If you play with "Complex Vehicles enabled" and no other Vehicle is available in that Nation's BR, then you are allowed to switch to a different Country.
